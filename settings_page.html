▶ Gmail App Password Guide
1. Go to myaccount.google.com/security
2. Enable 2-Step Verification if not already active
3. Go to App passwords
4. Select "Mail" and your device, then click Generate
5. Copy the 16-character password and paste it above
6. IMAP must be enabled in Gmail Settings > Forwarding and POP/IMAP
▶ Outlook / Microsoft 365 App Password Guide
1. Go to account.activedirectory.microsoft.com
2. Sign in with your Microsoft 365 account
3. Go to Security info > Add method > App password
4. Name it (e.g. "SENTINEL") and click Add
5. Copy the generated password and paste it above
6. Ensure Basic Authentication is enabled for IMAP in your org admin
